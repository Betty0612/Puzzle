About Memory Puzzle

Memory Puzzle is a game where player will find a pair for each card. When the game begins, all pictures are hidden. The player flips over two cards. If they match, then they stay overturned. Otherwise, they flip back. The player needs to overturn all the cards in the fewest moves to win.

If you want to start a new game please click PLAY button!

How to play Memory Puzzle?

  1. Select two cards to find the same pictures.
  2. If you match the pictures you can try to find other pair.
  3. If cards don't match they flip back.
  4. The player that overturns all the cards in the fewest moves wins!
